# Phase 4 - STL in C++

This phase is focused on learning and practicing **STL (Standard Template Library)** in C++.

The main goal of this phase is to understand how to use built-in C++ containers, iterators, and algorithms to write cleaner, faster, and more efficient code.

---

## Overview

In this phase, I am learning the most important parts of STL that are commonly used in:

* problem solving
* DSA
* coding interviews
* competitive programming

This phase is important because STL helps reduce manual code and provides optimized data structures and functions.

Instead of writing everything from scratch, STL allows me to solve problems in a smarter and faster way.

---

## Topics Covered

### 1. STL Basics

* `pair`
* iterators
* common STL functions

### 2. Sequence Containers

* `vector`
* `list`
* `deque`

### 3. Container Adapters

* `stack`
* `queue`
* `priority_queue`

### 4. Ordered Associative Containers

* `set`
* `multiset`
* `map`
* `multimap`

### 5. Unordered Associative Containers

* `unordered_set`
* `unordered_multiset`
* `unordered_map`
* `unordered_multimap`

### 6. STL Algorithms / Utility Functions

* `sort()`
* `reverse()`
* `find()`
* `count()`
* `binary_search()`
* `lower_bound()`
* `upper_bound()`
* `min_element()`
* `max_element()`
* `accumulate()`
* `next_permutation()`
* `__builtin_popcount()`

## Practice Approach

In this phase, I am following this structure:

* **one STL topic = one folder**
* **one question = one `.cpp` file**
* first understand syntax
* then understand operations
* then solve related questions
* then revise with practice

This helps in making STL usage natural instead of just memorizing functions.
