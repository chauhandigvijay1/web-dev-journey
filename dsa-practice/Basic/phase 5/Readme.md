# Phase 5 - Basic Recursion

This phase focuses on understanding **recursion fundamentals** in C++.

The main goal of this phase is to build:

* recursive thinking
* understanding of function call flow
* base condition handling
* problem decomposition skills

---

## Overview

Recursion is one of the most important concepts in DSA.

It helps in solving problems by breaking them into **smaller subproblems**.

This phase builds the foundation required for:

* backtracking
* divide and conquer
* dynamic programming
* advanced DSA problems

---

## Topics Covered

This phase includes the following recursion problems:

* Understand recursion (print something N times)
* Print name N times
* Print 1 to N using recursion
* Print N to 1 using recursion
* Sum of first N numbers
* Factorial of a number
* Reverse an array
* Check if string is palindrome
* Fibonacci number

## Practice Approach

In this phase, I am following this structure:

* **one problem = one `.cpp` file**
* first understand the recursive idea
* then identify base case
* then identify recursive call
* then dry run the recursion
* then implement

This helps in understanding recursion deeply instead of memorizing patterns.
