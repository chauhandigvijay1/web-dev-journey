# Phase 1 - C++ Basics

This phase contains my beginner-level C++ learning and practice.

It includes the core concepts of C++ along with topic-wise practice questions to build a strong programming foundation.

---

## Overview

In this phase, I focused on learning the basic syntax and logic of C++.

This phase helped me understand how to write simple programs, use conditions and loops, work with arrays and strings, and break problems into smaller parts using functions.

---

## Topics Covered

### 1. Input / Output

* `cin`
* `cout`

### 2. Basics of C++

* `using namespace std`
* datatypes
* `getline`
* basic syntax

### 3. Conditional Statements

* `if`
* `if-else`
* `if-else-if`
* nested `if-else`
* `switch case`

### 4. Arrays

* 1D Arrays
* 2D Arrays
* traversal
* indexing

### 5. Strings

* basic string handling
* input
* indexing
* common operations

### 6. Loops

* `for` loop
* `while` loop
* `do-while` loop
* nested loops
* `break` and `continue`

### 7. Functions

* function basics
* parameters and arguments
* return type
* function prototype
* pass by value
* pass by reference
* function overloading
* scope
