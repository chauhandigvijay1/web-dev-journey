## Overview

Hashing is one of the most important concepts in DSA.

It is widely used in:

* arrays
* strings
* frequency based problems
* lookup based problems
* optimization problems

This phase introduces the basic idea of hashing through a small set of highly important beginner problems.

This phase includes the following hashing problems:

* Basic Hashing
