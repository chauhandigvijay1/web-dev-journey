# Phase 6 - Basic Hashing

This phase focuses on learning the basics of **hashing** in C++.

The main goal of this phase is to understand how to:

* count frequencies efficiently
* store repeated information
* reduce brute force approaches
* solve lookup based problems faster

---

## Overview

Hashing is one of the most important concepts in DSA.

It is widely used in:

* arrays
* strings
* frequency based problems
* lookup based problems
* optimization problems

This phase introduces the basic idea of hashing through a small set of highly important beginner problems.

---

## Topics Covered

This phase includes the following hashing problems:

* Basic Hashing
* Counting Frequencies of Array Elements
* Highest Occurring Element in an Array
