# 📝 Internship Application Form (Tailwind CSS)

This project is a modern internship application form built using HTML and Tailwind CSS.

It demonstrates form handling UI, layout design, and responsive styling.

---

## 🎯 Objective

The goal of this project is to:

- Practice building structured forms  
- Learn Tailwind CSS styling  
- Create clean and user-friendly UI  
- Understand form inputs and layouts  

---

## ✨ Features

- Fully responsive design  
- Clean input fields with focus effects  
- Radio buttons and checkboxes  
- Dropdown selections  
- Modern gradient button  
- Glassmorphism container UI  

---

## 🛠️ Tech Stack

- HTML5  
- Tailwind CSS (CDN)  
- No JavaScript  

---

## 📁 Project Structure

internship-form-css/

├── index.html

## ▶️ How to Run Locally

### Method 1: Direct Open

- Open `index.html`  
- It will run in your browser  

---

### Method 2: VS Code (Recommended)

1. Open folder in VS Code  
2. Install **Live Server Extension**  
3. Right-click `index.html`  
4. Click **Open with Live Server**
