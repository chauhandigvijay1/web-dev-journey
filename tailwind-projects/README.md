This repository contains modern UI projects built using **HTML and Tailwind CSS**.

These projects focus on creating clean, responsive, and visually appealing user interfaces without using JavaScript.

---

## 🎯 Objective

The goal of this repository is to:

- Learn and apply Tailwind CSS utility-first approach  
- Build modern UI designs (glassmorphism, gradients, cards)  
- Improve layout, spacing, and responsiveness  
- Create professional frontend projects

## ▶️ How to Run Locally

### Method 1: Direct Open

- Open any project folder  
- Double-click on the HTML file  
- It will run in your browser  

---

### Method 2: VS Code (Recommended)

1. Open project folder in VS Code  
2. Install **Live Server Extension**  
3. Right-click on HTML file  
4. Click **Open with Live Server**
