# 🌐 Personal Portfolio Website (Tailwind CSS)

This is my personal portfolio website built using HTML and Tailwind CSS.  
It showcases my skills, projects, experience, and contact information in a clean and modern UI.

---

## 🎯 Objective

The goal of this project is to:

- Build a professional developer portfolio  
- Practice Tailwind CSS and responsive design  
- Showcase my projects and technical skills  
- Create a strong online presence for recruiters  

---

## ✨ Features

- Modern UI with glassmorphism effect  
- Responsive design (mobile-first)  
- Smooth hover animations  
- Clean section-based layout  
- Contact form UI  

---

## 🛠️ Tech Stack

- HTML5  
- Tailwind CSS (CDN)  
- No JavaScript  

---

## 📁 Project Structure

portfolio-css/

├── index.html  
├── images/  
└── README.md  

---

## ▶️ How to Run Locally

### Method 1: Direct Open

- Open `index.html`  
- It will run in your browser  

---

### Method 2: VS Code (Recommended)

1. Open folder in VS Code  
2. Install **Live Server Extension**  
3. Right-click `index.html`  
4. Click **Open with Live Server**
