# 🚀 Modern Landing Page (Tailwind CSS)

This project is a modern, responsive landing page built using **HTML and Tailwind CSS**.

It features a clean UI, glassmorphism elements, and a premium dark theme using gradient colors.

---

## 🎯 Objective

The goal of this project is to:

- Build a modern landing page UI  
- Practice Tailwind CSS utility-first styling  
- Improve layout, spacing, and responsiveness  
- Create a recruiter-ready frontend project  

---

## ✨ Features

- Responsive Navbar (CSS-only mobile menu)  
- Hero section with call-to-action buttons  
- Feature cards with glass effect  
- Pricing section  
- Contact form UI  
- Clean footer  
- Fully responsive design  

---

## 🛠️ Tech Stack

- HTML5  
- Tailwind CSS (CDN)  
- No JavaScript

## ▶️ How to Run Locally

### Method 1: Direct Open

- Open `index.html`  
- It will run in your browser  

---

### Method 2: VS Code (Recommended)

1. Open project folder in VS Code  
2. Install **Live Server Extension**  
3. Right-click on `index.html`  
4. Click **Open with Live Server**
