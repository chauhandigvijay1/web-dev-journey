# 📝 Internship Application Form

This is a basic HTML project that represents an internship application form.

The goal of this project is to practice creating structured forms using semantic HTML elements.

---

## 🎯 Features

- Input fields (Name, Email, etc.)
- Form validation using HTML attributes
- Clean and simple structure
- Beginner-friendly layout
